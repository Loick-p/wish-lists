# Wish Lists

Application pour créer des listes de cadeaux pour votre famille ou vos amis.

## Stack
**Back :** PHP/Laravel

**Front :** Tailwind, Preline

## Environnement Back

- PHP 8.4
- Laravel 12
- PostgreSQL 18

## Roadmap
- Création du front avec React
- Adapation du back en API
- Adapation du front avec l'API
